GitGuardian-Uploader

Overview

GitGuardian-Uploader is a desktop application that streamlines the process of creating GitHub repositories with proper licensing and documentation protection.

Purpose

This tool was developed to help developers who frequently need to:

  • Create new GitHub repositories with proper licensing
  • Protect their intellectual property
  • Ensure consistent repository structure
  • Save time on repository setup

Requirements

  • Python 3.7+
  • GitHub Personal Access Token
  • Internet connection

Installation

git clone https://github.com/yourusername/GitGuardian-Uploader
cd GitGuardian-Uploader
python requirements.py

Usage

  1. Run the application: python protect_code_gui.py
  2. Enter your GitHub token
  3. Select your project directory
  4. Fill in repository details
  5. Choose a license
  6. Click "Upload to GitHub"

Use Case Example

Sarah, a software developer, has created a new machine learning library. She needs to:

  • Upload it to GitHub
  • Protect it with proper licensing
  • Include patent notices
  • Create proper documentation
Instead of doing all this manually, she uses GitGuardian-Uploader to handle everything in one go.
